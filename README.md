## Репозиторий содержит материалы командного проекта.

- `demo.html` — гайд по стилям и их использованию. Используется как эталон для всех участников команды. Внутри гайд по тому, как оформлять заголовки, текст, кнопки и цвета. Все участники проекта должны придерживаться этих правил при вёрстке страниц
- `styles.css` — основной файл стилей, подключаемый ко всем страницам проекта. Содержит CSS-переменные, шрифты, базовые классы и кнопки
- `README.md` — файл с описанием проекта
- `fonts/...` — шрифты  
- `[...].html` — страницы макета
